操作

在浏览器中访问[https://code.visualstudio.com](https://code.visualstudio.com/)，点击下载按钮。 你将会得到一个`.deb`文件，在终端中使用命令 `dpkg` 安装该软件包。

```shell
$ sudo dpkg -i ./code_xxx.deb
```
